# Smart Notes - Django Web Application

Smart Notes is a full-stack web application built using Django.  
It allows authenticated users to create and manage personal notes through a dashboard interface.

---

##  Features

- User authentication (Login / Logout)
- Custom user model with roles
- Add notes
- Delete notes
- Persistent data using SQLite
- Dashboard interface
- Clean UI with HTML, CSS, and JavaScript

---

##  Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

## ⚙️ How To Run This Project

1. Clone the repository:
https://github.com/choegyaljr/smart-notes-django.git

2. Navigate into the project folder:
cd smart-notes-django 

3. Install dependencies:
pip install -r requirements.txt


4. Apply migrations: 
python manage.py migrate


5. Run the development server:
python manage.py runserver


6. Open in browser:
http://127.0.0.1:8000/
